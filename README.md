# AndroidAPP-ABB-
一个使用react-native做的安卓app，IOS运行部分功能有问题

使用react-native 0.61制作，使用了antd-rn,react-native-tcp-socket,react-native-storage,
需要翻墙工具+JDK1.8才能运行

安装模块
yarn  
然后
yarn add react-native-cli -g

运行
react-native run-android（安卓手机需要打开开发者模式）
